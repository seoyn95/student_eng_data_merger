# Student ENG Data Merger

이 프로그램은 학원 업무 중 불편했던 데이터를 자동으로 읽고 병합하기 위해 개인적으로 만든 스크립트 기록입니다.  

## 기능
- 여러 개의 데이터 파일(워드프로세서) 읽기
- 데이터를 분석하여 하나의 파일로 병합
- 파일 구조가 다를 경우 간단한 에러 처리 + 학생 이름 오류 시 에러 처리

## 사용법
1. 로컬에 파일들을 모두 같은 폴더에 넣습니다.
2. `weekly/subject_merge_program.py` 실행
3. 병합된 파일은 output 폴더에 날짜 적힌 파일로 생성됩니다.
4. 엑셀 파일의 경우 pdf 변환기에 가깝습니다.

## 주의
- 학원 업무용으로 만든 개인 스크립트입니다.
- 개인 학습 및 편의용으로만 사용한 개인 기록입니다.
